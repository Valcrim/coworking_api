# Описание
Веб-приложение на fastapi для бронирования комнат в коворкинге.

Работа была сделана в рамках [тестового задания](https://shiftproject.ru/resources/uploads/SHIFT_Python_Test_Task_2026_b952dd82c7.pdf) для проекта ШИФТ.


# Как запустить
1. Проект собирается с помощью Docker следующей командой:
```
docker compose up
```

2. Перейдите на http://localhost:8000/docs
